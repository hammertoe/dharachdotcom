I'm Matt Hamilton, and I've built two AI-powered applications in Barbados that showcase what's possible with Google's AI stack. These are the projects you're looking for.

Yuh Gettin' Tru?

an AI-enhanced aggregate search across 62 stores in Barbados. Search over 283,000 products to find exactly what you need, from hardware to homewares to chocolate.

The name comes from the Bajan phrase "Yuh gettin' tru?" meaning "Are you managing?" or "Are you getting through (your task)?" - because finding what you need in Barbados used to be a mission.

Tech Stack:

  • MongoDB Atlas Vector Search - semantic search across 283,000+ products
  • Gemini 2.5 Flash - generates embeddings and powers natural language understanding
  • Python / FastAPI - backend API
  • Google Cloud Run - serverless deployment

Yuh Hear Dem

an AI guide to Barbadian Parliament that helps citizens understand what's really being said in political debates and decisions. Making democracy more accessible, one question at a time.

The name comes from the Bajan phrase "Yuh hear dem?" meaning "Did you hear what they said?" - because parliamentary debates can be hard to follow and understand.

Tech Stack:

  • Gemini 2.5 Flash - powers the conversational AI and summarization
  • MongoDB Atlas Vector Search - semantic search across parliamentary transcripts
  • Python / FastAPI - backend API
  • Google Cloud Run - serverless deployment
  • Whisper - audio transcription of parliamentary sessions
